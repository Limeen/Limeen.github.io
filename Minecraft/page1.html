Rejoindre le serveur

Rejoindre avec Minecraft Java

1

Lancer Minecraft en 1.19.4

Utilisez votre launcher favoris pour lancer le jeu, vous pouvez utiliser TL Legacy qui est gratuit si vous en avez aucun.
2

Ajouter le serveur

Utilisez l'ip legominecraft.ddns.net pour vous connecter au serveur.
3

Créer votre compte en jeu

Une fois connecté au serveur, vous allez devoir créer votre compte avec la commande /register <mot de passe> <confirmer mot de passe> dans le chat.
