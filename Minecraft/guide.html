Authentification

Vous trouverez ici les commandes relié au système d'authentification.

  • Inscription
  • Connexion
  • Auto-Connexion
  • Désinscription

/register <mot de passe> <mot de passe une deuxième fois>
Cette commande permet de vous inscrire lors de votre première connexion sur le serveur pour empêcher les autres joueurs de se connecter avec votre pseudo.

/login <mot de passe>
Cette commande permet de vous connecter sur le serveur avec le mot de passe que vous avez défini lors de votre première connexion sur le serveur.

/remember
Cette commande permet de sauvegarder votre session sur votre adresse IP pendant une période de 5 jours, ce qui vous permet de vous connectez automatiquement sans entrer votre mot de passe à chaque fois. 

/unregister
Cette commande efface le mot de passe de votre compte et vous désinscrit du serveur.
Vous pouvez utiliser cette commande si vous voulez changer de mot de passe.

Tpa

Le /tpa permet de se téléporter sur un autre joueur avec son autorisation.

  • Téléportation
  • Accepter
  • Refuser

/tpa <pseudo>
Cette commande permet de vous téléporter sur quelqu'un avec son autorisation

/tpaccept
Cette commande permet d'accepter une demande de téléportation d'un autre joueur

/tpdeny
Cette commande permet de refuser une demande de téléportation d'un autre joueur

Home

Les homes vous permettent de sauvegarder des emplacements et de vous y re-téléporter quand vous le souhaitez.

  • Créer
  • Téléportation
  • Supprimer
  • Voir

/sethome <nom de votre home>
Cette commande permet de sauvegarder un emplacement sous le nom désiré

Vous pouvez créer jusqu'à 3 homes, et 6 si vous soutenez le serveur.

/home <nom du home>
Cette commande permet de vous téléporter à un home que vous avez créé

/delhome <nom du home>
Cette commande permet de supprimer un home que vous avez créé

/homes
Cette commande permet de voir la liste de vos homes avec leurs noms

Claim

Les claims permettent de protéger votre terrain des autres joueurs, ils expirent au bout de 30 jours d'inactivité sur le serveur.

  • Créer
  • Redimensionner
  • Supprimer
  • Permissions
  • Voir

/claim <rayon>
Cette commande permet de claim les blocs autour de vous en fonction du rayon donné

Vous pouvez également claim une surface en faisant clique droit sur deux extrémités avec une pelle en bois.

- Prenez un bâton en bois et faites clique droit sur votre claim pour voir les délimitations
- Avec la pelle en bois, faites clique droit sur le coin que vous souhaitez déplacer puis re-faites clique droit là ou vous voulez le mettre

/unclaim
Supprime le claim dans lequel vous vous trouvez

/abandonallclaims
Supprime tout vos claims

/trust <pseudo>
Cette commande permet d'autoriser un autre joueur de construire & d'intéragir avec les blocs sur le claim où vous vous trouvez

/untrust <pseudo>
Cette commande permet de supprimer les autorisations à un joueur défini de construire & d'intéragir sur le claim où vous vous trouvez

Les permissions sont indépendante de chaque claim, ce qui signifie que vous pouvez avoir 2 claims séparément et sur un seul des deux l'autre joueur à les permissions 

/claimlist
Affiche vos claims et le nombre de blocs que vous pouvez claim, vous gagnez des blocs en fonction de votre temps de jeu.

Trade

Le /trade vous permet de faire des échanges de blocs ou d'argent en toute sécurité avec d'autres joueurs.

  • Échanger
  • Accepter
  • Refuser
  • Activer/Désactiver 

/trade <pseudo>
Cette commande permet d'envoyer une demande d'échange à un autre joueur

/trade accept
Cette commande permet d'accepter une demande d'échange d'un autre joueur

/trade deny
Cette commande permet de refuser une demande d'échange d'un autre joueur

/trade toggle
Cette commande permet de désactiver/activer les demandes d'échange des autres joueurs

Shop & Économie

Vous trouverez ici toutes les commande relié à l'économie du serveur.

  • Argent
  • Classement
  • Échanger
  • Shop: acheter
  • Shop: vendre

/money bal
/money bal <pseudo>
Cette commande permet de voir votre compte en banque ou celui d'un autre joueur

/money top
Cette commande permet de voir le classement des joueurs les plus riches

Vous pouvez échanger de l'argent grâce au /trade.

/shop
Permet de voir le shop et d'acheter des choses en naviguant dans les menus 

/shop
Vous pouvez utiliser le /shop pour vendre en navigant dans les menus, cette fonctionnalité n'est pas comptatible pour les joueurs Minecraft: Bedrock Édition.

/sellgui
Ouvre un interface ou vous pouvez glisser toutes vos ventes. Fermer l'interface pour tout vendre.

Tombe

Lorsque vous mourrez une tombe apparaît pendant 24 heures pour éviter que votre stuff despawn.

  • Voir
  • Tête

/deathchest
Cette commande permet de voir où se trouvent vos dernières tombes

Lorsque vous mourrez, vous avez 20% de chance que votre tête soit dropé.

Lune de sang

Les lunes de sang apparaissent de façon aléatoire et rendent la nuit beaucoup plus difficile.

  • Effets
  • Phénomènes 
  • Boss

- Les monstres ont x2 plus de vie
- Les monstres donnent des effets
- L'expérience dropé par les monstres est multiplié par x5
- Les monstres dropent du charbon, des lingots de fer/or de façon aléatoire

Les lunes de sang affectent l'overworld avec de nombreux événements paranormaux

- Brouillard intense
- Orage permanent
- Bruits étranges permanent 
- Apparition de hordes de monstres sur des joueurs aléatoire
- Apparition d'un Boss
- Insomnie permanente, personne ne peut dormir
- Remplacement de torche aléatoire en torche de redstone
- Disparition de blocs aléatoire
- Disparition d'animaux aléatoire
- Transformation de villageois aléatoire en sorcière
- Transformation de creeper aléatoire en creeper super-chargé
- Apparition de structures étranges
- Herobrine

Lorsqu'il y a une Lune de sang, un boss surpuissant apparaît aléatoirement sur quelqu'un.
Ce boss loot x3 Pomme de Notch
