Guide du serveur

Authentification

Vous trouverez ici les commandes relié au système d'authentification

  • Inscription
  • Connexion
  • Auto-Connexon
  • Désinscription

/register <mot de passe> <mot de passe une deuxième fois>

Cette commande permet de vous inscrire lors de votre première connexion sur le serveur pour empêcher les autres joueurs de se connecter avec votre pseudo.

/login <mot de passe>

Cette commande permet de vous connecter sur le serveur avec le mot de passe que vous avez défini lors de votre première connexion sur le serveur.

/remember

Cette commande permet de sauvegarder votre session sur votre adresse IP pendant une période de 5 jours, ce qui vous permet de vous connectez automatiquement sans entrer votre mot de passe à chaque fois. 

/unregister

Cette commande efface le mot de passe de votre compte et vous désinscrit du serveur.
Vous pouvez utiliser cette commande si vous voulez changer de mot de passe.
